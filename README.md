sql-beginner-project
**Overview**
A beginner-friendly project that analyses daily expenses using SQL and automates expense entry using n8n.
**Tools Used**
SQL (SQLite)
n8n (Automation)
Google Sheets
GitHub
**What This Project Does**
Analyses expense data using basic SQL queries
Calculates total and category-wise spending
Automatically saves new expense entries to Google Sheets using n8n
**Key Concepts**
SQL: SELECT, SUM, GROUP BY, MAX
Automation: Webhook → Set → Google Sheets
Clean project structure and documentation
**Purpose**
Built to understand SQL fundamentals, basic automation, and real-world data flow at an entry level.

Level: Beginner / Entry-Level
